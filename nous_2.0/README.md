# Nous Clinical Tech | Intelligent SEO System

![Nous Clinical Tech](public/og-image.jpg)

**Nous** is a high-performance clinical SEO interface built with modern WebGL and React technologies. It combines high-end 3D visuals with analytical precision for technical data environments.

## 🚀 Key Features
- **Central Neural Link (3D Orb)**: Real-time glass refraction and chromatic aberration.
- **Topographic Data Grid**: High-performance instanced rendering of 1200+ data nodes.
- **Cinematic Camera Rig**: Intelligent transitions and mouse-parallax depth.
- **Clinical Aesthetics**: Minimal surgery-inspired UI with high-contrast support.
- **GPU-Aware Tiering**: Automatic performance scaling based on hardware capability.
- **Asset Sychronization**: Progressive system loading with a cinematic "System_Init" screen.

## 🛠 Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **3D Engine**: React Three Fiber / Three.js
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion & React Spring
- **State Management**: Zustand
- **Calibration**: Leva (in dev mode)

## 📦 Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Production Build
```bash
npm run build
npm start
```
